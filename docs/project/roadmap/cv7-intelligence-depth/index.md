# CV7 — Intelligence Depth

**Status:** Planned
**Goal:** Improve memory quality and intelligence depth after the runtime identity model, onboarding path, and extension boundaries are mature enough to support broader adoption.

---

## What This Is

The former CV6 theme moves here. Once Mirror Mind is cleaner as a multi-user
framework and runtime-relevant identity behavior is fully database-backed, the
next capability can return to the quality of what the mirror remembers and how
it reasons over that memory.

Likely areas:
- better extraction prompts
- richer Jungian layering, including shadow-oriented work where appropriate
- reinforcement tuning
- smarter hybrid search and retrieval weighting
- improved memory signal quality vs. noise

---

## Status

Not yet planned in detail.

CV7 planning should start from the more mature multi-user/runtime foundation
established by CV0–CV6.
